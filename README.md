# Dotfiles

Helix and Alacritty, for now.

On Linux:

```bash
git clone https://github.com/brunomdsmartins/dotfiles ~/.config/dotfiles

cd ~/.config/dotfiles

chmod +x install.sh
./install.sh
```

On Windows:

```powershell
git clone https://github.com/brunomdsmartins/dotfiles $HOME\AppData\Roaming\dotfiles

cd $HOME\.config\dotfiles

.\install.ps1
```
